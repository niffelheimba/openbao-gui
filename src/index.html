YUBIKEY

Request YubiKey certificate

For hardware-backed mTLS certificates. Requires YubiKey Manager CLI (ykman) to be installed and a YubiKey with PIV enabled.

The PIN and management key are used only for this request and are not saved. Each profile can choose a slot below; the default selector is kept for compatibility. If the management key field is blank, the app supplies the standard default YubiKey PIV management key so ykman will not prompt invisibly. Touch prompts are shown by the YubiKey blinking rather than a Windows dialog.

YubiKey issuance is planned but not implemented in this build.
